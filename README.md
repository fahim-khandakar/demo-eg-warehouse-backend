##HEP Software built for organizations managing part inventories, orders, and inter-partner operations. It includes:

## Feature

- [x] Role-Based Access**: Supports multiple roles like admin, user, super admin, and principal for tailored access.
- [x] Part Request Portal**: Partners can directly request parts through a Portal.
- [x] Inventory Management**: Full visibility and control over parts through bufferIn/ buffer out tracking, availability monitoring, and historical logs.
- [x] Loan System**: Parts can be loaned to partners, with status tracking (Open/Close).
- [x] Proof of Delivery (PoD) & Shipment**: Principals can manage and track shipments sent to partners with detailed receipts.
- [x] Comprehensive Analytics**: Dashboards for order status, part availability, loan records, and user performance insights.
- [x] Custom Power-Based Permissions**: Users receive granular access and actions based on predefined powers.
- [x] User Activity Tracking**: Logs every action with IP, entity, and metadata for auditing and transparency.
- [x] Dynamic Dashboard ** : User based Dynamic Dashboard
- [x] Excel/PDF Report Generator** :   Export shipment reports, inventory logs, user activity, etc., in Excel formats.
